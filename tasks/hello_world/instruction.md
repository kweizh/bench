# Wasp Hello World

## Background
In Wasp, you define pages and routes in the `.wasp` file. A `page` declaration specifies the React component to render, and a `route` declaration specifies the path.

## Requirements
- Create a new page named `HelloPage` that renders a React component displaying the text "Hello Wasp!".
- Define a route named `HelloRoute` that maps the path `/hello` to the `HelloPage`.
- The React component should be exported from `src/HelloPage.tsx`.

## Implementation
1. The project is already initialized at `/home/user/my-app`.
2. Open `main.wasp` and add the `page` and `route` declarations.
3. Create `src/HelloPage.tsx` with the React component.
4. Ensure the app is running.

## Output
- Project path: /home/user/my-app
- Start command: wasp start
- Port: 3000
