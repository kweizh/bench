# Cancel a Triggered Knock Workflow Before Email Delivery ## Background Knock (https://knock.app) is a notification infrastructure platform whose workflow engine orchestrates cross-channel messaging from a single trigger. Knock supports **workflow cancellation**: if a triggered workflow contains a step that pauses execution (such as a `delay`), the workflow run can be aborted before the downstream channel steps deliver any messages. To enable this, the trigger request must include a `cancellation_key`, and a subsequent `POST /v1/workflows/{key}/cancel` request must reference the same `cancellation_key`. ## Implementation Details

The script performs the following operations:

1. **Upserts a workflow** in the Knock development environment with:
   - A 30-second delay function step
   - A Mailtrap email channel step (channel_key: `mailtrap`)
   - Sender email: `sender-<run-id>@<MAILTRAP_DOMAIN>`
   - Subject and body templates that render the recipient's name

2. **Activates the workflow** in the development environment

3. **Triggers the workflow** exactly once with:
   - A single inline recipient with id `user-<run-id>`
   - Recipient email: `<GMAIL_USER_NAME>+receiver-<run-id>@gmail.com`
   - Recipient name: `User <run-id>`
   - Cancellation key: `cancel-<run-id>`

4. **Waits 5 seconds** to ensure the trigger is processed

5. **Cancels the workflow** using the same workflow key and cancellation key

6. **Logs the results** to `/home/user/cancel_task/output.log`:
   - `Workflow Key: cancel-flow-<run-id>`
   - `Workflow Run ID: <uuid>`
   - `Cancellation Key: cancel-<run-id>`
   - `Recipient Email: <GMAIL_USER_NAME>+receiver-<run-id>@gmail.com`

## Usage

1. Install dependencies: `npm install`
2. Create a `.env` file with the required environment variables (see `.env.example`)
3. Run the automation: `npm start`

All resource identifiers are suffixed with the run-id to prevent collisions between concurrent runs, and the cancellation is issued well before the 30-second delay elapses, ensuring the email is never delivered.
