## What was built

### `run.py` — Dataflow pipeline (5 steps)

```
FileSource  →  map(parse)  →  stateful(deduplicate)  →  map(format)  →  FileSink
```

1. **`op.input`** — reads `input.jsonl` line-by-line via `FileSource`.
2. **`op.map("parse")`** — deserialises each JSON line and returns `(user_id, event_dict)` to key the stream by `user_id`.
3. **`op.stateful("deduplicate")`** — applies `DeduplicateLogic` per `user_id`:
   - **State**: `dict[event_id → datetime of last emitted occurrence]`
   - **`on_item`**: first prunes all `event_id`s whose last-seen timestamp is older than 10 s from the current event (memory cleanup), then applies the deduplication rule — drops the event if `current_ts ≤ last_ts + 10s`, otherwise emits it and updates the state.
   - **`snapshot`**: returns a `deepcopy` of the state dict for safe pickling.
4. **`op.map("format")`** — strips the internal `_ts` field and serialises back to a JSON string, keeping the `(key, value)` tuple structure that `FileSink` requires.
5. **`op.output`** — writes surviving events to `output.jsonl` via `FileSink`.
