## How it works

```
FileSource  →  parse JSON line  →  key_on(user_id)
    →  collect_window(EventClock + SessionWindower(gap=30min))
    →  map_value(summarize → JSON string)
    →  FileSink(sessions.jsonl)
```

Key design decisions:

| Decision | Rationale |
|---|---|
| `EventClock(wait_for_system_duration=0)` | Batch job — EOF triggers watermark flush, no real-time waiting needed |
| `SessionWindower(gap=timedelta(minutes=30))` | Closes a window after 30 min of inactivity per user |
| `op.map_value` instead of `op.map` | Keeps the stream keyed (`(user_id, str)`) as required by `FileSink` (a `FixedPartitionedSink`) |
| `OUTPUT_PATH.touch()` | `FileSink` requires the file to pre-exist before writing |
