## What was built

### `/home/user/graph_pipeline/pipeline.py`
A single-file Bytewax 0.21.1 dataflow with:

1. **`UnionFind` class** — path-compressed, union-by-size disjoint-set structure stored as two plain `dict`s, making it fully picklable for Bytewax's SQLite recovery.

2. **`union_find_mapper`** — stateful_map callback that:
   - Lazily creates the UnionFind state on first invocation
   - Parses each edge JSON, calls `union(u, v)`
   - Emits `merged` + `new_component_size` when components are joined
   - Emits `already_connected` + `component_size` when they're already in the same component

3. **Dataflow topology**:
   ```
   FileSource(input_edges.jsonl)
     → key_on("singleton")        # constant key → single partition
     → stateful_map(union_find)   # Union-Find logic
     → map_value(identity)        # drop routing key
     → FileSink(output_events_<ZEALT_RUN_ID>.jsonl)
   ```

### Recovery
SQLite recovery partition pre-initialised at `recovery/part-0.sqlite3`.
