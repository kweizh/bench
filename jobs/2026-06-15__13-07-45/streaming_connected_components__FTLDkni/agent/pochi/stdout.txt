- `FixedPartitionedSource` with a single partition reading `input_edges.jsonl`
- Supports recovery via `snapshot()` returning the current line index

### Output Sink (`JSONLSink`)
- `FixedPartitionedSink` writing JSON objects line-by-line to `output_events_${ZEALT_RUN_ID}.jsonl`

### Dataflow Logic
1. Reads edges from `input_edges.jsonl`
2. Routes all edges to a single partition using `op.key_on` with a constant key `"global"` — ensuring the Union-Find has a global view
3. `op.stateful_map` maintains the `UnionFind` state, emitting either `{"status": "merged", "new_component_size": N}` or `{"status": "already_connected", "component_size": N}`
4. Sinks results to the output file
