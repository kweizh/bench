**`user_post_counts`** (view collection)
```sql
SELECT users.id       AS id,
       users.name     AS username,
       COUNT(posts.id) AS post_count
FROM users
LEFT JOIN posts ON posts.author = users.id
GROUP BY users.id
```
Fields auto-derived: `id` (text), `username` (text), `post_count` (number).

> Note: PocketBase 0.31's built-in `users` auth collection stores the display name in the `name` column (not `username`), so the view aliases it as `username` to satisfy the API contract.
