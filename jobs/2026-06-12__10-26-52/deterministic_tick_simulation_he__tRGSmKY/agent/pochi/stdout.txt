## Key Design Decisions

- **`HeadlessApplicationConfiguration.updatesPerSecond = 0`** тАФ runs the main loop as fast as possible without wall-clock throttling.
- **`CountDownLatch`** тАФ used instead of `Thread.join()` since `HeadlessApplication` doesn't expose a `join()` method. The latch is counted down in `dispose()`, guaranteeing the output file is written before the main thread proceeds.
- **Symplectic Euler integration** тАФ velocity is updated before position each tick (`vx += ax*dt`, `vy += ay*dt`, `x += vx*dt`, `y += vy*dt`).
- **Deterministic** тАФ uses `dt` from the config file, never `Gdx.graphics.getDeltaTime()`.
- **`Gdx.app.exit()`** called from `render()` when `currentTick >= totalTicks`, triggering `pause()` + `dispose()` asynchronously.
- **Output written in `dispose()`** тАФ ensures the file is flushed before the latch signals completion.
