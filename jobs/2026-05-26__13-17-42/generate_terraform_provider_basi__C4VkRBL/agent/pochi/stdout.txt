### Usage example (Terraform HCL)

```hcl
terraform {
  required_providers {
    petstore = {
      source = "registry.terraform.io/petstore/petstore"
    }
  }
}

provider "petstore" {
  server_url = "http://api.petstore.com/v1"
}

resource "petstore_pet" "doggie" {
  name       = "doggie"
  photo_urls = ["https://example.com/doggie.jpg"]
}
```

After `terraform apply`, the `id` attribute will be populated from the server's response.
