Because Express query values arrive as strings, the schema MUST coerce strings to numbers using ArkType morphs (a single declarative pipeline — no `JSON.parse`, `parseInt`, or other hand-written coercion). * Implement a reusable middleware factory `validate(source, schema)` where `source` is either `'body'` or `'query'`.
