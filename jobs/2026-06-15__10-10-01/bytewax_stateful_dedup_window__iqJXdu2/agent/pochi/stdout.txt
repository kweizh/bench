## Implementation Summary

**`run.py`** implements a Bytewax dataflow that:

1. **Reads** JSON objects from `input.jsonl` using `FileSource`
2. **Parses** each line into a Python dict with an ISO 8601 timestamp converted to a `datetime` object
3. **Keys** the stream by `user_id` using `key_on`
4. **Deduplicates** using `StatefulLogic`:
   - `DedupLogic` maintains a `Dict[str, datetime]` mapping `event_id` → last emitted timestamp
   - On each event, stale entries (older than 10s from the current event's timestamp) are cleaned up
   - An event is emitted only if its `event_id` has not been seen within the last 10 seconds (inclusive: `current_ts <= last_ts + 10s`)
   - If emitted, the state is updated with the new timestamp
5. **Formats** results back to JSON strings (stripping the internal `_ts` field)
6. **Writes** deduplicated events to `output.jsonl` using `FileSink`
