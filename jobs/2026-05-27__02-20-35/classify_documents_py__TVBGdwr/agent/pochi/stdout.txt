## What the script does

1. **Parses args** – single positional `file` argument via `argparse`.
2. **Validates** the file exists and `LLAMA_CLOUD_API_KEY` is set; exits non-zero otherwise.
3. **Uploads** the PDF with `client.files.create(file=<fh>, purpose="classify")`, obtaining a `file_id`.
4. **Classifies** via `client.classifier.classify(file_ids=[file_id], rules=[...], mode="FAST")` — this convenience method submits the job, polls until done, and returns a `JobGetResultsResponse`.
5. **Emits** a single JSON line to stdout:
   ```json
   {"file": "/absolute/path/to/doc.pdf", "type": "invoice", "confidence": 0.97}
   ```
6. **Exits 0** on success; exits with distinct non-zero codes (1–6) for each failure category.
