### `run.py` – CLI entrypoint

**Flow:**
1. `argparse` validates that `--pdf`, `--config`, and `--output` are all supplied; missing flags print usage and exit 2.
2. `get_api_key()` reads `LLAMA_CLOUD_API_KEY`; absent/empty → stderr message + exit 1.
3. `validate_pdf()` checks the file exists and has a `.pdf` extension.
4. `load_and_validate_config()` parses the JSON categories file and verifies every entry has `name` + `description`.
5. `upload_pdf()` calls `client.files.create(file=..., purpose="split")` and returns the `file_id`.
6. `run_split_job()` calls `client.beta.split.split(document_input={"type": "file_id", "value": file_id}, categories=categories)`, which creates the job and polls until completion, then returns `result.result.segments`.
7. `serialise_segments()` maps each `SplitSegmentResponse` to `{category, pages, confidence_category}`.
8. `write_output()` JSON-dumps the payload (pretty-printed, 2-space indent) and creates any missing parent directories.
