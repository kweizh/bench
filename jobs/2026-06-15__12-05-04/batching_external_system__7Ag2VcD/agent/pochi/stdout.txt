## Implementation — `/home/user/bytewax_batching/dataflow.py`

The pipeline follows these steps:

1. **Input** — `FileSource` reads `input.jsonl` line-by-line.
2. **Parse** — `op.map` deserialises each line with `json.loads`.
3. **Filter** — `op.filter` drops any record where `metric_value < 0` (3 records removed: -5.0, -1.0, -10.0).
4. **Key** — `op.key_on` partitions the stream by `device_id`.
5. **Collect** — `op.collect` batches per key with `max_size=10` and `timeout=timedelta(seconds=1)`.
6. **Output** — A custom `SQLiteSink` (`DynamicSink` + `StatelessSinkPartition`) receives each `(device_id, [metrics])` batch and calls `executemany` for bulk inserts.

The `ZEALT_RUN_ID` env var (`zr-7ag2vcd`) is used to name the database file.
