### What the script does

1. **Input** — reads `user_events.jsonl` line-by-line via `FileSource`.
2. **Parse** — each JSON line is parsed; the `timestamp` string is converted to an aware UTC `datetime` (stored as `_dt`), and the event is keyed by `user_id`.
3. **EventClock** — uses the `_dt` field on each event as the event-time timestamp. `wait_for_system_duration=0` is appropriate for a bounded/batch file source: when the file reaches EOF the watermark advances past all open windows, flushing them.
4. **SessionWindower(gap=30min)** — a session closes when no event arrives within 30 minutes of the previous one (per user key).
5. **collect_window** — gathers all events for each session into a time-ordered list.
6. **Summarize** — maps each completed window to the required JSON structure, computing `session_start` / `session_end` from the min/max timestamps and `event_count` from the list length.
7. **Output** — writes one JSON object per line to `sessions.jsonl` via `FileSink`.
