## Endpoints

### `POST /api/uploads/presign`
- Protected by `apis.RequireAuth("users")` → returns 401 if unauthenticated
- Generates a UUID key, mints a 300-second AWS SigV4 presigned PUT URL (path-style: `http://127.0.0.1:9000/uploads/<key>?X-Amz-Signature=…`)
- Creates a `pending_upload` record tied to the authenticated user
- Returns `{ "url": "…", "key": "…", "expiresAt": "…" }` (RFC3339, ~300s ahead)

### `POST /api/uploads/finalize`
- Protected by `apis.RequireAuth("users")` → returns 401 if unauthenticated
- Accepts `{ "key": "…" }`
- Returns 404 if `pending_upload` row doesn't exist or belongs to another user
- Returns 404 if S3 `HeadObject` fails (object not yet uploaded)
- On success: deletes the `pending_upload` row, creates an `uploads` row, returns `{ "key": "…" }`
