The same scope must also define a composite `Order` schema that references the custom keywords by bare name: ``` Order = { id: slug, customerPhone: usPhone, cardNumber: creditCard, total: number > 0 } ``` A CLI entry point reads one JSON payload from stdin and validates it against `Order`. function luhnCheck(num: string) {
    let sum = 0;
    let shouldDouble = false;
    for (let i = num.length - 1; i >= 0; i--) {
        let digit = parseInt(num.charAt(i), 10);
        if (shouldDouble) {
