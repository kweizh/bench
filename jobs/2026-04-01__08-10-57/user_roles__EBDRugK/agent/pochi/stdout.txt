In this task, you will implement role-based access control (RBAC) to restrict access to an Admin page. ## Requirements - **Authentication**: Enable `usernameAndPassword` authentication. - **Data Model**: - `User` entity with a `role` field (String, can be "ADMIN" or "USER", default: "USER"). - **Custom Signup**: Implement `userSignupFields` to allow setting the `role` during signup for testing purposes (e.g., if username is "admin", set role to "ADMIN"). - **Pages**: - `MainPage` (protected): Accessible to any logged-in user.
