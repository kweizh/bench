4. Runs a TypeScript snippet via `sandbox.process.codeRun(snippet)` that computes `factorial(6)` and `console.log`s the result.
5. Extracts the first meaningful output line (skipping npm notices the sandbox runtime may inject), then writes `Factorial: <value>` to `output.log`.
6. In the `finally` block, calls `daytona.delete(sandbox)` regardless of success or failure.
7. Exits with code 1 if `exitCode !== 0`.
