## How It Works

1. **TanStack Table** (`useReactTable`) renders a data grid with 4 initial users (columns: ID, Name, Email, Role, Actions).
2. Each row has an **Edit** button. Clicking it sets `editingId` in state, which causes the row to render as an `EditRowForm` component instead of a regular display row.
3. **TanStack Form** (`useForm`) manages the inline edit form with:
   - `defaultValues` pre-populated from the current row data
   - `validators.onSubmit` performing validation (Name required, Email required + format check)
   - `onSubmit` handler that updates the users array and exits edit mode
4. The form uses `<form.Field>` components for each field (Name, Email input; Role dropdown), with error display via `field.state.meta.errors`.
5. **Save** triggers `form.handleSubmit()` — validation errors prevent save, valid data updates the row.
6. **Cancel** calls `setEditingId(null)` to discard changes and exit edit mode.
