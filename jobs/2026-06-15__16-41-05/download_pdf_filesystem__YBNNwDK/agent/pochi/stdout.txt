The Capacitor Filesystem plugin gives apps a NodeJS-like API for working with the device's storage. On the web target, this plugin transparently falls back to a persistent IndexedDB-backed store so that the same JavaScript code that runs on iOS and Android also works in the browser-based preview that this task uses for verification.
