## Architecture

- **Vite + React + TypeScript** project running on port **8432**
- **TanStack Query** (`useQuery`) fetches products from a mock async API (`src/api/products.ts`) with 8 sample products
- **TanStack Router** manages all cart state via URL search parameters (`?cart=...`) using `validateSearch` with Zod schema validation
- Cart state is serialized as JSON in the URL (e.g., `?cart={"1":2,"3":1}` = product 1 qty 2, product 3 qty 1)

## How Cart State Works

1. Cart state is parsed from `search.cart` (URL search param) into `Record<string, number>`
2. Adding/removing items or changing quantities calls `navigate()` with updated serialized cart
3. The URL updates immediately (e.g., `/?cart={"2":1,"5":3}`)
4. Refreshing the page restores cart state from the URL
