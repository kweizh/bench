A separate `SubViewport` containing its own `Camera3D` renders a small 3D world; a HUD `TextureRect` displays that `SubViewport`'s `ViewportTexture` in real time, like a CCTV monitor. Repositioning the source camera at runtime must change what the monitor shows on the next rendered frame. ## Requirements Deliver a Godot 4 project that the verifier can load headlessly. The project must contain a main scene that contains all required nodes wired together, and a script (attached to the scene root) exposing the public API listed below.
