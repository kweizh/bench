## What was implemented

### `src/roundTrip.ts`
- **`exportJsonSchema(schema: Type): object`** â€” calls `schema.toJsonSchema({ target: "draft-2020-12" })` and ensures the `$schema: "https://json-schema.org/draft/2020-12/schema"` field is present in the returned object.
- **`roundTrip(schema: Type): Type`** â€” exports the schema to JSON Schema via `exportJsonSchema`, strips the `$schema` meta-key (not understood by the parser), then re-parses using `jsonSchemaToType` from `@ark/json-schema`, returning the reconstructed ArkType `Type`.

### `src/cli.ts`
- Reads a JSON document `{ "schema": ..., "corpus": [...] }` from stdin.
- Builds the original `Type` via `type.raw(input.schema)`.
- Calls `exportJsonSchema` and `roundTrip` from `./roundTrip`.
- For each corpus item, checks `original.allows(item)` and `roundTripped.allows(item)` and records whether they agree.
- Writes `{ jsonSchema, agreements, allAgree }` to stdout as a single JSON document.
- Exits with code `0` when `allAgree` is `true`, otherwise exits with `1`.
