# Cross-Session Memory CLI with Vercel AI SDK + Alchemyst (TypeScript) ## Background You will build a small Node.js TypeScript CLI that uses the Vercel AI SDK (`ai` package) wrapped with Alchemyst's `withAlchemyst` middleware from `@alchemystai/aisdk` to give the LLM persistent, cross-session memory. Alchemyst is a Context Engine that gives LLMs durable memory keyed by `(userId, sessionId)`. When `withAlchemyst(...)` wraps `generateText`, every call to the wrapped function automatically (a) retrieves past memories for the `userId`, (b) injects them into the prompt, and (c) persists the new turn. This lets the agent remember a preference established in one session and reuse it in a completely different session for the same user. The CLI must demonstrate this with two phases: - `--phase establish`: the user tells the assistant a personal dietary preference. The assistant acknowledges and the memory is written. - `--phase recall`: a *new session id* is used, the user asks an open question, and the assistant must answer using the preference that was stored in the previous phase. ## Requirements - A Node.js project (npm) at `/home/user/vercel-ai-memory` written in TypeScript. - Production dependencies that **must** be declared in `package.json`: - `ai` (Vercel AI SDK) - `@ai-sdk/openai` (Vercel AI SDK OpenAI provider) - `@alchemystai/aisdk` (Alchemyst middleware) - TypeScript build that emits the runnable entrypoint to `dist/main.js`. - The CLI accepts a single argument `--phase <establish|recall>` and reads: - `ALCHEMYST_AI_API_KEY` (Alchemyst API key) - `OPENAI_API_KEY` (OpenAI API key) - `ZEALT_RUN_ID` (run id used to namespace the user and session ids; see Parallel-Run Safety below) - Use the wrapped function `withAlchemyst(generateText, { apiKey: process.env.ALCHEMYST_AI_API_KEY })` and call it with an OpenAI model from `@ai-sdk/openai` (for example `openai('gpt-4o-mini')` or another small OpenAI chat model). **Do not** call OpenAI directly without the Alchemyst wrapper. - Every call to the wrapped `generateText` MUST pass both `userId` and `sessionId`. The middleware will return `MISSING_PARAMETERS` if either is omitted. - Print the model's response text to **stdout** (nothing else on stdout other than the model output is required, but it must include the response text). - No mocks, stubs, fake clients, or hard-coded model replies: the program must perform real network calls to Alchemyst and OpenAI. ## Parallel-Run Safety (REQUIRED) This task may run multiple times concurrently against the same Alchemyst account. import { generateText } from 'ai';
import { openai } from '@ai-sdk/openai';
import { withAlchemyst } from '@alchemystai/aisdk';
import * as dotenv from 'dotenv';

dotenv.config();

async function main() {
  const ALCHEMYST_AI_API_KEY = process.env.ALCHEMYST_AI_API_KEY;
  const OPENAI_API_KEY = process.env.OPENAI_API_KEY;
  const ZEALT_RUN_ID = process.env.ZEALT_RUN_ID;

  if (!ALCHEMYST_AI_API_KEY || !OPENAI_API_KEY || !ZEALT_RUN_ID) {
    console.error('Missing required environment variables (ALCHEMYST_AI_API_KEY, OPENAI_API_KEY, ZEALT_RUN_ID)');
    process.exit(1);
  }

  const args = process.argv.slice(2);
  const phaseIndex = args.indexOf('--phase');
  if (phaseIndex === -1 || phaseIndex + 1 >= args.length) {
    console.error('Missing --phase argument');
    process.exit(1);
  }

  const phase = args[phaseIndex + 1];
  if (phase !== 'establish' && phase !== 'recall') {
    console.error('Invalid phase. Must be establish or recall');
    process.exit(1);
  }

  const userId = `vercel-memory-user-${ZEALT_RUN_ID}`;
  const sessionId = `${phase}-${ZEALT_RUN_ID}`;

  const wrappedGenerateText = withAlchemyst(generateText, {
    apiKey: ALCHEMYST_AI_API_KEY,
  });

  const model = openai('gpt-4o-mini');

  if (phase === 'establish') {
    const prompt = "Please remember this about me: I am vegan and I am allergic to peanuts. Acknowledge that you will remember.";
    const { text } = await wrappedGenerateText({
      model,
      prompt,
      userId,
      sessionId,
    } as any);
    console.log(text);
  } else if (phase === 'recall') {
    const prompt = "Based on what you remember about my dietary restrictions, what should I avoid at a dinner party? List the exact dietary label(s) I told you.";
    const { text } = await wrappedGenerateText({
      model,
      prompt,
      userId,
      sessionId,
    } as any);
    console.log(text);
  }
}

main().catch((err) => {
  console.error(err);
  process.exit(1);
});
