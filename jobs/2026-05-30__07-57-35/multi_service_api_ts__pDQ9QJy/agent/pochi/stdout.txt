In this task, you will create a multi-service Encore application in TypeScript and deploy it to Encore Cloud. ## Requirements * Create an Encore application named `myapp`. * Create a `user` service with an API endpoint `GET /user/:id` that returns a user object `{"id": number, "name": string}`.
