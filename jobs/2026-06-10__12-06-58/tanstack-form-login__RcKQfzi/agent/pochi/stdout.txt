## How it works

**TanStack Form** (`useForm`) manages field state with `defaultValues` for `email` and `password`. Each `form.Field` has a `validators.onChange` handler that runs on every keystroke.

**Zod** schemas validate each field:
- `emailSchema` — `z.string().email(...)` — requires a valid email format
- `passwordSchema` — `z.string().min(8, ...)` — requires ≥ 8 characters

**Error display** — `field.state.meta.errors[0]` is rendered below each input inside a `role="alert"` paragraph, and the input gets a red border via the `input-error` class.

**Success state** — `onSubmit` sets `loginSuccess = true`, which swaps the form for a success card displaying "Login successful".
