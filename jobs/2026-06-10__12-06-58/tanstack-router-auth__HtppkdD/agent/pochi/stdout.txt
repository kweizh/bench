**Key implementation details:**

- **Auth context** (`src/auth.tsx`): `AuthProvider` wraps the app and exposes `isAuthenticated`, `login()`, and `logout()` via `useAuth()`.
- **Router context** (`src/main.tsx`): The `App` component reads `isAuthenticated` from `useAuth()` and passes it into `RouterProvider` as `context`, so the router re-evaluates `beforeLoad` guards on every auth state change.
- **Route protection** (`src/routes/dashboard.tsx`): Uses `beforeLoad: ({ context }) => { if (!context.isAuthenticated) throw redirect({ to: '/login' }) }` — unauthenticated visits to `/dashboard` are redirected to `/login`.
- **Login flow**: Clicking "Login" calls `login()` then navigates to `/dashboard`.
- **Logout flow**: Clicking "Logout" calls `logout()` then navigates to `/login`.
- **Vite plugin**: `TanStackRouterVite()` auto-generates `src/routeTree.gen.ts` with all three routes registered.
