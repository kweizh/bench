### Key implementation details

**URL-based cart state** — The cart is encoded as a JSON array in `?cart=[{"id":1,"qty":2},...]`. The route uses `validateSearch` with a Zod schema to parse/validate the search params on load, so refreshing the page always restores the full cart.

**TanStack Query** — `useQuery({ queryKey: ['products'], queryFn: fetchProducts })` fetches the product list with a simulated 600ms delay, showing a spinner during loading.

**TanStack Router navigation** — Every cart mutation (add, remove, quantity change, clear, category filter) calls `navigate({ search: prev => ({ ...prev, cart: updater(prev.cart) }), replace: true })`, keeping the full URL state in sync without adding browser history entries.
