The state machine is implemented at `/home/user/myproject` with the following files:

**`src/types.ts`** — defines all ArkType schemas and the validated transition function:
- `State` — discriminated union on `status` with four branches: `Idle`, `Loading` (with `startedAt: integer >= 0`), `Success` (with `data: unknown`, `fetchedAt: integer >= 0`), `Failure` (with `code: integer [400,599]`, `reason: string [1,200]`)
