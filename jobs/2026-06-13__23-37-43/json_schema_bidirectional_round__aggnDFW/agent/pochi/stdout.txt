- **`exportJsonSchema(schema: Type) => object`** â€” calls `schema.toJsonSchema({ dialect: "https://json-schema.org/draft/2020-12/schema" })`, which always produces a JSON Schema document with the `$schema` draft-2020-12 URI field.
