## How slug generation works

### `models/Article.js`

```js
hooks: {
  // Single-record path: fires BEFORE validation so the slug is present
  // before the notNull constraint on the slug column is checked.
  beforeValidate(article) {
    if (article.title) {
      article.slug = generateSlug(article.title);   // "My Title" → "my-title"
    }
  },

  // Bulk path: bulkCreate() skips per-instance hooks by default,
  // so beforeBulkCreate is used to iterate the whole batch explicitly.
  beforeBulkCreate(articles) {
    for (const article of articles) {
      if (article.title) article.slug = generateSlug(article.title);
    }
  },
}
```
