**Associations** (key to avoiding the "Include unexpected" error)
```js
Mail.belongsTo(Person, { as: 'sender',   foreignKey: 'senderId' });
Mail.belongsTo(Person, { as: 'receiver', foreignKey: 'receiverId' });
```
Each alias is declared on the `Mail` model with a distinct foreign key, so Sequelize knows exactly which column to join on for each alias.
