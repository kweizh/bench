**Why the Super Many-to-Many direct associations matter:**  
Sequelize's `through.include` option is not supported. Instead, the Super Many-to-Many pattern (direct `hasMany`/`belongsTo` on `Enrollment`) lets you nest `Enrollment` as a regular `include` on `Course`, which in turn holds the nested `Semester`. This is the approach used in the query:

```js
Student.findOne({
  where: { name: 'Alice' },
  include: [
    {
      model: Course,
      include: [
        {
          model: Enrollment,      // direct association via hasMany/belongsTo
          include: [{ model: Semester }],
        },
      ],
    },
  ],
});
```
