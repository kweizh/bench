## Endpoints

### `POST /seed`
- Calls `sequelize.sync({ force: true })` to wipe and recreate tables
- Inserts companies → departments → employees recursively from the JSON body
- Returns `200 { message: "Database seeded successfully." }`

### `GET /companies/filtered`
Uses two nested `include` blocks with **`required: false`** (LEFT OUTER JOIN) so that:

| Scenario | Behaviour |
|---|---|
| Company has **no active** departments | Still returned — `departments: []` |
| Active department has **no senior** employees | Still returned — `employees: []` |
| Inactive departments (`status !== 'active'`) | Excluded |
| Non-senior employees (`role !== 'senior'`) | Excluded |
