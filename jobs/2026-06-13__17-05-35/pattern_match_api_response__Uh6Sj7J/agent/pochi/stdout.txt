## Implementation Details

The handler uses ArkType's `match` API with the fluent `.at("status")` pattern to discriminate on the `status` property:

```ts
const handleResponse = match
  .at("status")
  .match({
    "'success'": (o) => `OK:${JSON.stringify(o.data)}`,
    "'error'": (o) => `ERR ${o.code} ${o.reason}`,
    "'pending'": () => "PENDING",
    default: "assert",
  });
```

The entrypoint reads a single JSON object from stdin, passes it through the matcher, and writes the formatted result to stdout. On any error (including unmatched input, which throws due to `default: "assert"`), the process exits with code 1.
