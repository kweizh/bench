The tool reads a small text-based scripting language describing 2D convex polygons and queries, then evaluates each query using libGDX math primitives (notably `com.badlogic.gdx.math.Intersector` and `com.badlogic.gdx.math.Polygon`) inside a `HeadlessApplication` and writes one result per query to standard output. No OpenGL context, no native windowing, and no audio must be initialized. ## Requirements * Implement a Gradle-based libGDX project at `/home/user/myproject` that runs entirely with the headless backend (`com.badlogicgames.gdx:gdx-backend-headless:1.14.2`). * Pin the libGDX version to `1.14.2` (core `com.badlogicgames.gdx:gdx:1.14.2`). * Wire an `ApplicationListener` (or `ApplicationAdapter`) that runs under `new HeadlessApplication(listener, config)` with `config.updatesPerSecond = 0` so the program ticks once and exits. * Read the path to the script file from the first program argument (forwarded via `--args`). * For each non-empty, non-comment line in the script, execute exactly one of the commands below and print exactly one output line to stdout.
