The tool reads a small text-based scripting language describing 2D convex polygons and queries, then evaluates each query using libGDX math primitives (notably `com.badlogic.gdx.math.Intersector` and `com.badlogic.gdx.math.Polygon`) inside a `HeadlessApplication` and writes one result per query to standard output. ## How it works

| Component | Detail |
|-----------|--------|
| **Runtime** | `HeadlessApplication` with `config.updatesPerSecond = 0`; all work done in `create()` |
| **Threading** | `CountDownLatch` in `GeomListener` lets `main()` block until `create()` finishes and `Gdx.app.exit()` is called â€” ensures stdout is fully flushed before the JVM exits |
| **Script reading** | Plain `java.io.BufferedReader` over the absolute path â€” works with or without a GDX `Files` implementation |
| **POLY** | Builds a `com.badlogic.gdx.math.Polygon` from the vertex array; stored in a `HashMap<String,Polygon>` |
| **OVERLAP** | `Intersector.overlapConvexPolygons(polyA, polyB)` |
| **CONTAINS** | `Intersector.isPointInPolygon(float[], offset, count, x, y)` on the polygon's transformed vertices; input coordinate tokens echoed verbatim |
| **SEGMENTS** | `Intersector.intersectSegments(Vector2,Vector2,Vector2,Vector2,Vector2)` with `Locale.ROOT` / `%.3f` formatting |
| **AREA** | `Polygon.area()` with `Locale.ROOT` / `%.3f` formatting |
| **Errors** | Any unknown command, undefined polygon, or malformed number prints `ERROR <1-based-lineNo> <reason>` and continues |
