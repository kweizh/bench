## Key Design Decisions

- **Fixed timestep**: Uses `1.0/60.0` as `deltaTime` for every `engine.update()` call — never reads `Gdx.graphics.getDeltaTime()`.
- **Headless thread joining**: `Main` polls `Thread.getAllStackTraces()` for the `"HeadlessApplication"` thread and `join()`s it to ensure the JVM doesn't exit before output is printed.
- **Duplicate print guard**: A `printed` boolean flag prevents `printFinalState()` from being called more than once, since `render()` may be invoked after `Gdx.app.exit()` is scheduled.
- **Locale.ROOT formatting**: All floats use `String.format(Locale.ROOT, "%.3f", value)` with explicit `-0.000` handling.
- **Entity ordering**: Entities are stored in insertion order so output matches the scenario file order.
- **No LWJGL**: Only `gdx-backend-headless` is on the classpath; no desktop GL backends.
