Unlike the unified API base (`https://unify.apideck.com`), file uploads and downloads must hit a separate host (`https://upload.apideck.com`) and the executor must send the file's metadata as a JSON string in the `x-apideck-metadata` HTTP header while the request body holds the raw binary content.
