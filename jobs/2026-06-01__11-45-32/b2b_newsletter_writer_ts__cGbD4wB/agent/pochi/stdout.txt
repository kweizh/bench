# B2B Newsletter Writer (TypeScript CLI) ## Background Inspired by Alchemyst AI's community B2B Newsletter Writer template, build a Node.js TypeScript command-line tool that uses the Alchemyst AI context engine together with the OpenAI Chat Completions API to produce a short, topic-focused newsletter for a B2B audience. The tool must: 1. Ingest a fixed corpus of seeded research articles into the Alchemyst context engine. 2. Take a `--topic` argument from the CLI, query the context engine for the most relevant snippets, and use OpenAI to compose a 3-section newsletter that draws on the retrieved context. 3. Persist the generated newsletter as Markdown to a known output path so a downstream verifier can inspect it.
