**Hook file:** `/home/user/myproject/pb_hooks/sanitize_email.pb.js`

```js
onRecordCreateRequest((e) => {
    if (e.collection.name === "users") {
        var email = e.record.get("email");
        if (typeof email === "string" && email.length > 0) {
            e.record.set("email", email.trim().toLowerCase());
        }
    }
    e.next();
});
```

**How it works:**
- `onRecordCreateRequest` fires before every record-create API call and receives the in-flight event `e`.
- The guard `e.collection.name === "users"` ensures the hook only touches the `users` collection; `_superusers` and any other auth collections are left untouched.
- `e.record.get("email")` / `e.record.set("email", …)` read and rewrite the field before it hits the database.
- `e.next()` propagates the handler chain — omitting this would silently block all record creation.
