## `main.go` — Key Implementation Details

**PocketBase embedding** — `pocketbase.New()` creates the app, and `app.OnServe().BindFunc(...)` hooks into the serve lifecycle (the correct API for PocketBase v0.39.x, which replaced the old `OnBeforeServe`).

**Route registration** — `e.Router.GET("/api/s3-presign", handler)` registers the endpoint on PocketBase's router before it starts accepting connections.

**Handler logic:**
1. Reads `filename` from the query string — returns `400 Bad Request` (via `apis.NewBadRequestError`) if absent.
2. Reads `AWS_REGION`, `AWS_ACCESS_KEY_ID`, `AWS_SECRET_ACCESS_KEY`, `AWS_BUCKET`, and `ZEALT_RUN_ID` from the environment.
3. Constructs the S3 object key as `uploads/<ZEALT_RUN_ID>/<filename>`.
4. Initialises an AWS SDK v2 S3 client with static credentials.
5. Generates a presigned `PutObject` URL (valid for 15 minutes) via `s3.NewPresignClient`.
6. Returns `200 OK` with `{"url": "https://..."}`.
